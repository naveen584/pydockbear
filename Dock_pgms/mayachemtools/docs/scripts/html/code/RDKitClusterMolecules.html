  29 """
  30 RDKitClusterMolecules.py - Cluster molecules using 2D fingerprints
  31 
  32 Usage:
  33     RDKitClusterMolecules.py [--butinaSimilarityCutoff <number>]  [--butinaReordering <yes or no>]
  34                              [--clusteringMethod <Butina, Centroid, CLink...>]
  35                              [--fingerprints <MACCS166Keys, Morgan, PathLength...> ] [--infileParams <Name,Value,...>]
  36                              [--numClusters <number>] [--outfileMode <SingleFile or MultipleFiles>]
  37                              [ --outfileParams <Name,Value,...> ] [--overwrite] [--paramsFingerprints <Name,Value,...>]
  38                              [--similarityMetric <Dice, Tanimoto...>] [-w <dir>] -i <infile> -o <outfile> 
  39     RDKitClusterMolecules.py -h | --help | -e | --examples
  40 
  41 Description:
  42     Cluster molecules based on a variety of 2D fingerprints using Butina [ Ref 136 ] or any
  43     other available hierarchical clustering methodology and write them to output file(s).
  44 
  45     The default fingerprints types for various fingerprints are shown below:
  46 
  47         AtomPairs              IntSparseIntVect
  48         MACCS166Keys           ExplicitBitVect
  49         Morgan                 UIntSparseIntVect
  50         MorganFeatures         UIntSparseIntVect
  51         PathLength             ExplicitBitVect
  52         TopologicalTorsions    LongSparseIntVect
  53  
  54     The Dice and Tanimoto similarity functions available in RDKit are able to
  55     handle fingerprints corresponding to both IntVect and BitVect. All other
  56     similarity functions, however, expect BitVect fingerprints to calculate
  57     pairwise similarity. Consequently, ExplicitBitVect fingerprints, instead of
  58     default IntVect fingerprints, are generated for AtomPairs, Morgan,
  59     MorganFeatures, and TopologicalTorsions to calculate similarity.
  60 
  61     The supported input file formats are: Mol (.mol), SD (.sdf, .sd), SMILES (.smi,
  62     .txt, .csv, .tsv)
  63 
  64     The supported output file formats are: SD (.sdf, .sd), SMILES (.smi), CSV/TSV
  65     (.csv, .tsv, .txt)
  66 
  67 Options:
  68     -b, --butinaSimilarityCutoff <number>  [default: 0.55]
  69         Similarity cutoff to use during Butina clustering. The molecule pairs with
  70         similarity value greater than specified value or distance less than '1 - specified
  71         value' are considered neighbors. This value is only used during 'Butina' value
  72         of '-c, --clusteringMethod' option and determines the number of clusters
  73         during the clustering of molecules. It is ignored for all other clustering methods.
  74     --butinaReordering <yes or no>  [default: no]
  75         Update number of neighbors for unassigned molecules after creating a new
  76         cluster in order to insure that the molecule with the largest number of
  77         unassigned neighbors is selected as the next cluster center.
  78     -c, --clusteringMethod <Butina, Centroid, CLink...>  [default: Butina]
  79         Clustering method to use for clustering molecules. Supported values:
  80         Butina, Centroid, CLink, Gower, McQuitty, SLink, UPGMA, Ward.
  81         Butina is an unsupervised database clustering method to automatically
  82         cluster small and large data sets. All other clustering methods correspond
  83         to hierarchical clustering and require a priori specification of number of
  84         clusters to be generated.
  85     -f, --fingerprints <MACCS166Keys, Morgan, PathLength...>  [default: Morgan]
  86         Fingerprints to use for calculating similarity/distance between molecules.
  87         Supported values: AtomPairs, MACCS166Keys, Morgan, MorganFeatures, PathLength,
  88         TopologicalTorsions. The PathLength fingerprints are Daylight like fingerprints.
  89         The Morgan and MorganFeature fingerprints are circular fingerprints, corresponding
  90         Scitegic's Extended Connectivity Fingerprints (ECFP) and Features Connectivity
  91         Fingerprints (FCFP). The values of default parameters for generating fingerprints
  92         can be modified using '-p, --paramsFingerprints' option.
  93     -e, --examples
  94         Print examples.
  95     -h, --help
  96         Print this help message.
  97     -i, --infile <infile>
  98         Input file name.
  99     --infileParams <Name,Value,...>  [default: auto]
 100         A comma delimited list of parameter name and value pairs for reading 
 101         molecules from files. The supported parameter names for different file
 102         formats, along with their default values, are shown below:
 103             
 104             SD, MOL: removeHydrogens,yes,sanitize,yes,strictParsing,yes
 105             SMILES: smilesColumn,1,smilesNameColumn,2,smilesDelimiter,space,
 106                 smilesTitleLine,auto,sanitize,yes
 107             
 108         Possible values for smilesDelimiter: space, comma or tab.
 109     -n, --numClusters <number>  [default: 10]
 110         Number of clusters to generate during hierarchical clustering. This option is
 111         ignored for 'Butina' value of '-c, --clusteringMethod' option.
 112     -o, --outfile <outfile>
 113         Output file name.
 114     --outfileMode <SingleFile or MultipleFiles>  [default: SingleFile]
 115         Write out a single file containing molecule clusters or generate an individual file
 116         for each cluster. Possible values: SingleFile or MultipleFiles. The molecules are
 117         grouped for each cluster before they are written to output file(s) along with
 118         appropriate cluster numbers. The cluster number is also appended to output
 119         file names during generation of multiple output files.
 120     --outfileParams <Name,Value,...>  [default: auto]
 121         A comma delimited list of parameter name and value pairs for writing
 122         molecules to files. The supported parameter names for different file
 123         formats, along with their default values, are shown below:
 124             
 125             SD: compute2DCoords,auto,kekulize,no
 126             SMILES: kekulize,no,smilesDelimiter,space, smilesIsomeric,yes,
 127                 smilesTitleLine,yes
 128             
 129         Default value for compute2DCoords: yes for SMILES input file; no for all other
 130         file types. The kekulize and smilesIsomeric parameters are also used during
 131         generation of SMILES strings for CSV/TSV files.
 132     --overwrite
 133         Overwrite existing files.
 134     -p, --paramsFingerprints <Name,Value,...>  [default: auto]
 135         Parameter values to use for generating fingerprints. The default values
 136         are dependent on the value of '-f, --fingerprints' option. In general, it is a
 137         comma delimited list of parameter name and value pairs for the name of
 138         fingerprints specified using '-f, --fingerprints' option. The supported
 139         parameter names along with their default values for valid fingerprints
 140         names are shown below:
 141             
 142             AtomPairs: minLength,1 ,maxLength,30, useChirality,No
 143             Morgan:   radius,2, useChirality,No
 144             MorganFeatures:   radius,2, useChirality,No
 145             PathLength: minPath,1, maxPath,7, fpSize, 2048, bitsPerHash,2
 146             TopologicalTorsions: useChirality,No
 147             
 148     -s, --similarityMetric <Dice, Tanimoto...>  [default: Tanimoto]
 149         Similarity metric to use for calculating similarity/distance between molecules.
 150         Possible values: BraunBlanquet, Cosine, Dice, Kulczynski, RogotGoldberg,
 151         Russel, Sokal, Tanimoto.
 152     -w, --workingdir <dir>
 153         Location of working directory which defaults to the current directory.
 154 
 155 Examples:
 156     To cluster molecules using Butina methodology at a similarity cutoff of 0.55
 157     with automatic determination of number of clusters, Tanimoto similarity
 158     metric corresponding to Morgan fingerprints with radius of 2, and write out
 159     a single SMILES file containing clustered molecules along with cluster number
 160     for each molecule, type:
 161 
 162         % RDKitClusterMolecules.py  -i Sample.smi -o SampleOut.smi
 163 
 164     To cluster molecules using Butina methodology at similarity cutoff of 0.45
 165     with automatic determination of number of clusters, Dice similarity metric
 166     corresponding to Morgan fingerprints with radius of 2, and write out multiple
 167     SD files containing clustered molecules for each cluster, type:
 168 
 169         % RDKitClusterMolecules.py  -b 0.45 --outfileMode MultipleFiles
 170           -i Sample.smi -o SampleOut.sdf
 171 
 172     To cluster molecules using Ward hierarchical methodology to generate 15
 173     clusters, Dice similarity metric corresponding to Pathlength fingerprints with 
 174     path length between 1 and 7,  and write out a single TSV file for clustered
 175     molecules along with cluster numner for each molecule, type:
 176 
 177         % RDKitClusterMolecules.py  -c Ward -f PathLength -n 15
 178           -p 'minPath,1, maxPath,7' -i Sample.sdf -o SampleOut.tsv
 179 
 180     To cluster molecules using Centroid hierarchical methodology to generate 5
 181     clusters, Dice similarity metric corresponding to MACCS166Keys fingerprints
 182     for molecules in a SMILES CSV file, SMILES strings in column 1, name in
 183     column 2, and write out a single SD file for clustered molecules along with
 184     cluster numner for each molecule, type:
 185 
 186         % RDKitClusterMolecules.py  -c Centroid -f MACCS166Keys --infileParams
 187           "smilesDelimiter,comma,smilesTitleLine,yes,smilesColumn,1,
 188           smilesNameColumn,2" --outfileParams "compute2DCoords,yes"

 270 def ClusterMolecules():
 271     """Cluster molecules."""
 272 
 273     Mols = RetrieveMolecules()
 274     MolsFingerprints = GenerateFingerprints(Mols)
 275     MolsClusters = PerformClustering(Mols, MolsFingerprints)
 276     
 277     WriteMolecules(MolsClusters)
 278 
 279 def PerformClustering(Mols, MolsFingerprints):
 280     """Perform clustering."""
 281 
 282     ClusteredMols = []
 283     if re.match("^Butina$", OptionsInfo["ClusteringMethod"], re.I):
 284         return PerformButinaClustering(Mols, MolsFingerprints)
 285     else:
 286         return PerformHierarchicalClustering(Mols, MolsFingerprints)
 287     
 288     return ClusteredMols
 289 
 290 def PerformButinaClustering(Mols, MolsFingerprints):
 291     """Perform clustering using Butina methodology."""
 292 
 293     MiscUtil.PrintInfo("\nClustering molecules using Butina methodology and %s similarity metric..." % OptionsInfo["SimilarityMetric"])
 294     
 295     FingerprintsCount = len(MolsFingerprints)
 296     DistanceCutoff =  1 - OptionsInfo["ButinaSimilarityCutoff"]
 297     Reordering = OptionsInfo["ButinaReordering"]
 298     
 299     DistanceMatrix = GenerateLowerTriangularDistanceMatrix(MolsFingerprints)
 300 
 301     ClusteredMolIndices = Butina.ClusterData(DistanceMatrix, FingerprintsCount, DistanceCutoff, reordering = Reordering, isDistData = True)
 302 
 303     MolsClusters = []
 304     for Cluster in ClusteredMolIndices:
 305         MolsCluster = [Mols[MolIndex] for MolIndex in Cluster]
 306         MolsClusters.append(MolsCluster)
 307 
 308     return MolsClusters
 309 
 310 def PerformHierarchicalClustering(Mols, MolsFingerprints):
 311     """Perform hierarchical clustering."""
 312 
 313     try:
 314         import numpy
 315     except ImportError:
 316         MiscUtil.PrintError("Failed to import numpy python module. This is required to cluster molecules using hierarchical clustering methodology.")
 317     
 318     if OptionsInfo["NumClusters"] > len(Mols):
 319         MiscUtil.PrintError("The number of clusters, %d, specified using \"-n, --numClusters\" must be less than total number of valid molecules, %d" % (OptionsInfo["NumClusters"], len(Mols)))
 320     
 321     MiscUtil.PrintInfo("\nCluster molecules using %s hierarchical clustering methodology and %s similarity metric......" % (OptionsInfo["SpecifiedHierarchicalClusteringMethod"], OptionsInfo["SimilarityMetric"]))
 322     
 323     NumFingerprints = len(MolsFingerprints)
 324     NumClusters = OptionsInfo["NumClusters"]
 325     DistanceMatrix = GenerateLowerTriangularDistanceMatrix(MolsFingerprints)
 326     
 327     ClusterPicker = HierarchicalClusterPicker(OptionsInfo["SpecifiedHierarchicalClusteringMethodID"])
 328     ClusteredMolIndices = ClusterPicker.Cluster(numpy.asarray(DistanceMatrix), NumFingerprints, NumClusters)
 329 
 330     MolsClusters = []
 331     for Cluster in ClusteredMolIndices:
 332         MolsCluster = [Mols[MolIndex] for MolIndex in Cluster]
 333         MolsClusters.append(MolsCluster)
 334     
 335     return MolsClusters

 337 def WriteMolecules(MolsClusters):
 338     """Write out molecules for each cluster along with cluster numbers."""
 339 
 340     ClustersCount = len(MolsClusters)
 341     
 342     SingleOutFileMode = OptionsInfo["SingleOutFileMode"]
 343     TextOutFileMode = OptionsInfo["TextOutFileMode"]
 344     TextOutFileDelim = OptionsInfo["TextOutFileDelim"]
 345 
 346     Compute2DCoords = OptionsInfo["OutfileParams"]["Compute2DCoords"]
 347     
 348     SMILESIsomeric = OptionsInfo["OutfileParams"]["SMILESIsomeric"]
 349     SMILESKekulize = OptionsInfo["OutfileParams"]["Kekulize"]
 350     
 351     # Setup outfile names and writers...
 352     SetupClustersOutFilesNames(len(MolsClusters))
 353     SingleClusterWriter, ClustersOutfilesWriters = SetupMoleculeWriters(ClustersCount)
 354 
 355     MolCount = 0
 356     SingleMolClustersCount = 0
 357     
 358     if SingleOutFileMode:
 359         Writer = SingleClusterWriter
 360     
 361     for ClusterIndex in range(0, ClustersCount):
 362         MolsCluster = MolsClusters[ClusterIndex]
 363         ClusterNum = ClusterIndex + 1
 364 
 365         if len(MolsCluster) == 1:
 366             SingleMolClustersCount += 1
 367         
 368         if not SingleOutFileMode:
 369             Writer = ClustersOutfilesWriters[ClusterIndex]
 370             
 371         for Mol in MolsCluster:
 372             MolCount += 1
 373 
 374             if TextOutFileMode:
 375                 # Write out text file including SMILES file...
 376                 SMILES = Chem.MolToSmiles(Mol, isomericSmiles = SMILESIsomeric, kekuleSmiles = SMILESKekulize)
 377                 MolName = RDKitUtil.GetMolName(Mol, MolCount)
 378                 Line = TextOutFileDelim.join([SMILES, MolName, "%d" % ClusterNum])
 379                 Writer.write("%s\n" % Line)
 380             else:
 381                 # Write out SD file...
 382                 Mol.SetProp("ClusterNumber", "%s" % ClusterNum)
 383                 if Compute2DCoords:
 384                     AllChem.Compute2DCoords(Mol)
 385                 Writer.write(Mol)
 386     
 387     if SingleClusterWriter is not None:
 388         SingleClusterWriter.close()
 389     for ClusterOutfileWriter in ClustersOutfilesWriters:
 390         ClusterOutfileWriter.close()
 391 
 392     MiscUtil.PrintInfo("\nTotal number of clusters: %d" % ClustersCount)
 393 
 394     if ClustersCount > 0:
 395         MiscUtil.PrintInfo("\nNumber of clusters containing only a single molecule: %d" % SingleMolClustersCount)
 396         MiscUtil.PrintInfo("Average number of molecules per cluster: %.1f" % (MolCount/ClustersCount))
 397     
 398         MiscUtil.PrintInfo("\nNumber of molecules in each cluster:")
 399         MiscUtil.PrintInfo("ClusterNumber,MolCount")
 400         ClusterNum = 0
 401         for MolsCluster in MolsClusters:
 402             ClusterNum += 1
 403             MiscUtil.PrintInfo("%d,%d" % (ClusterNum, len(MolsCluster)))

 421 def GenerateFingerprints(Mols):
 422     """Generate fingerprints."""
 423 
 424     FingerprintsName = OptionsInfo["SpecifiedFingerprints"]
 425     
 426     MolsFingerprints = []
 427     if re.match("^AtomPairs$", FingerprintsName, re.I):
 428         return GenerateAtomPairsFingerprints(Mols)
 429     elif re.match("^MACCS166Keys$", FingerprintsName, re.I):
 430         return GenerateMACCS166KeysFingerprints(Mols)
 431     elif re.match("^Morgan$", FingerprintsName, re.I):
 432         return GenerateMorganFingerprints(Mols)
 433     elif re.match("^MorganFeatures$", FingerprintsName, re.I):
 434         return GenerateMorganFeaturesFingerprints(Mols)
 435     elif re.match("^PathLength$", FingerprintsName, re.I):
 436         return GeneratePathLengthFingerprints(Mols)
 437     elif re.match("^TopologicalTorsions$", FingerprintsName, re.I):
 438         return GenerateTopologicalTorsionsFingerprints(Mols)
 439     else:
 440         MiscUtil.PrintError("Fingerprints name, %s, is not a valid name" % FingerprintsName)
 441     
 442     return MolsFingerprints

 444 def GenerateAtomPairsFingerprints(Mols):
 445     """Generate AtomPairs fingerprints."""
 446 
 447     MiscUtil.PrintInfo("\nGenerating AtomPairs fingerprints...")
 448     
 449     MinLength = OptionsInfo["FingerprintsParams"]["AtomPairs"]["MinLength"]
 450     MaxLength = OptionsInfo["FingerprintsParams"]["AtomPairs"]["MaxLength"]
 451     UseChirality = OptionsInfo["FingerprintsParams"]["AtomPairs"]["UseChirality"]
 452 
 453     if OptionsInfo["GenerateBitVectFingerints"]:
 454         # Generate ExplicitBitVect fingerprints...
 455         FPSize = 2048
 456         BitsPerHash = 4
 457         MolsFingerprints = [rdMolDescriptors.GetHashedAtomPairFingerprintAsBitVect(Mol, minLength = MinLength, maxLength = MaxLength, includeChirality = UseChirality, nBits = FPSize, nBitsPerEntry = BitsPerHash) for Mol in Mols]
 458     else:
 459         # Generate IntSparseIntVect fingerprints...
 460         MolsFingerprints = [rdMolDescriptors.GetAtomPairFingerprint(Mol, minLength = MinLength, maxLength = MaxLength, includeChirality = UseChirality) for Mol in Mols]
 461 
 462     return MolsFingerprints

 489         MolsFingerprints = [rdMolDescriptors.GetMorganFingerprint(Mol, Radius, useFeatures = UseFeatures, useChirality = UseChirality) for Mol in Mols]

 548 def GenerateLowerTriangularDistanceMatrix(MolsFingerprints):
 549     """Generate a lower triangular distance matrix without the diagonal."""
 550 
 551     SimilarityFunction = OptionsInfo["SimilarityFunction"]
 552 
 553     DistanceMatrix = []
 554     NumFPs = len(MolsFingerprints)
 555     for Index1 in range(0, NumFPs):
 556         for Index2 in range(0, Index1):
 557             Distance =  1 - SimilarityFunction(MolsFingerprints[Index1], MolsFingerprints[Index2],)
 558             DistanceMatrix.append(Distance)
 559 
 560     return DistanceMatrix

 672 def ProcessSpecifiedFingerprintsParameters():
 673     """Process specified fingerprints parameters."""
 674 
 675     if re.match("^auto$", OptionsInfo["ParamsFingerprints"], re.I):
 676         # Nothing to process...
 677         return
 678 
 679     SpecifiedFingerprintsName = OptionsInfo["SpecifiedFingerprints"]
 680     
 681     # Parse specified fingerprints parameters...
 682     ParamsFingerprints = re.sub(" ", "", OptionsInfo["ParamsFingerprints"])
 683     if not ParamsFingerprints:
 684         MiscUtil.PrintError("No valid parameter name and value pairs specified using \"-p, --paramsFingerprints\" option corrresponding to fingerprints %s." % (SpecifiedFingerprintsName))
 685 
 686     ParamsFingerprintsWords = ParamsFingerprints.split(",")
 687     if len(ParamsFingerprintsWords) % 2:
 688         MiscUtil.PrintError("The number of comma delimited paramater names and values, %d, specified using \"-p, --paramsFingerprints\" option must be an even number." % (len(ParamsFingerprintsWords)))
 689 
 690     # Setup a canonical parameter names for specified fingerprints...
 691     ValidParamNames = []
 692     CanonicalParamNamesMap = {}
 693     for ParamName in sorted(OptionsInfo["FingerprintsParams"][SpecifiedFingerprintsName]):
 694         ValidParamNames.append(ParamName)
 695         CanonicalParamNamesMap[ParamName.lower()] = ParamName
 696 
 697     # Validate and set paramater names and value...
 698     for Index in range(0, len(ParamsFingerprintsWords), 2):
 699         Name = ParamsFingerprintsWords[Index]
 700         Value = ParamsFingerprintsWords[Index + 1]
 701 
 702         CanonicalName = Name.lower()
 703         if  not CanonicalName in CanonicalParamNamesMap:
 704             MiscUtil.PrintError("The parameter name, %s, specified using \"-p, --paramsFingerprints\" option for fingerprints, %s, is not a valid name. Supported parameter names: %s" % (Name, SpecifiedFingerprintsName, " ".join(ValidParamNames)))
 705 
 706         ParamName = CanonicalParamNamesMap[CanonicalName]
 707         if re.match("^UseChirality$", ParamName, re.I):
 708             if not re.match("^(Yes|No|True|False)$", Value, re.I):
 709                 MiscUtil.PrintError("The parameter value, %s, specified using \"-p, --paramsFingerprints\" option for fingerprints, %s, is not a valid value. Supported values: Yes No True False" % (Value, SpecifiedFingerprintsName))
 710             ParamValue = False
 711             if re.match("^(Yes|True)$", Value, re.I):
 712                 ParamValue = True
 713         else:
 714             ParamValue = int(Value)
 715             if ParamValue <= 0:
 716                 MiscUtil.PrintError("The parameter value, %s, specified using \"-p, --paramsFingerprints\" option for fingerprints, %s, is not a valid value. Supported values: > 0" % (Value, SpecifiedFingerprintsName))
 717         
 718         # Set value...
 719         OptionsInfo["FingerprintsParams"][SpecifiedFingerprintsName][ParamName] = ParamValue
 720 
 721 def ProcessSimilarityMetricParameter():
 722     """Process specified similarity metric value."""
 723 
 724     SimilarityInfoMap = {}
 725     CanonicalNameMap = {}
 726     
 727     for SimilarityFunctionInfo in DataStructs.similarityFunctions:
 728         Name = SimilarityFunctionInfo[0]
 729         Function = SimilarityFunctionInfo[1]
 730         
 731         SimilarityInfoMap[Name] = Function
 732         CanonicalName = Name.lower()
 733         CanonicalNameMap[CanonicalName] = Name
 734     
 735     SpecifiedCanonicalName = OptionsInfo["SimilarityMetric"].lower()
 736     SimilarityFunction = None
 737     if  SpecifiedCanonicalName in CanonicalNameMap:
 738         SimilarityName = CanonicalNameMap[SpecifiedCanonicalName]
 739         SimilarityFunction = SimilarityInfoMap[SimilarityName]
 740     else:
 741         MiscUtil.PrintError("Similarity metric name, %s, is not a valid name. " % OptionsInfo["SimilarityMetric"])
 742         
 743     OptionsInfo["SimilarityMetric"] = SimilarityName
 744     OptionsInfo["SimilarityFunction"] = SimilarityFunction
 745 
 746     # RDKit similarity functions, besides Dice and Tanimoto, are not able to handle int bit vectors...
 747     GenerateBitVectFingerints = False
 748     if not re.match("^(Tanimoto|Dice)$", SimilarityName, re.I):
 749         GenerateBitVectFingerints = True
 750     OptionsInfo["GenerateBitVectFingerints"] = GenerateBitVectFingerints
