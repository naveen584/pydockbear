  29 """
  30 RDKitPickDiverseMolecules.py - Pick a diverse subset of molecules
  31 
  32 Usage:
  33     RDKitPickDiverseMolecules.py [--clusteringMethod <Centroid, CLink...>]
  34                                  [--fingerprints <MACCS166Keys, Morgan, PathLength...>]
  35                                  [--infileParams <Name,Value,...>] [--mode <MaxMin or HierarchicalClustering>]
  36                                  [--numMols <number>]  [--outfileParams <Name,Value,...>] 
  37                                  [--overwrite] [--paramsFingerprints <Name,Value,...>]
  38                                  [--similarityMetric <Dice, Tanimoto...>] [-w <dir>] -i <infile> -o <outfile> 
  39     RDKitPickDiverseMolecules.py -h | --help | -e | --examples
  40 
  41 Description:
  42     Pick a subset of diverse molecules  based on a variety of 2D fingerprints using
  43     MaxMin [ Ref 135 ] or an available hierarchical clustering methodology and write
  44     them to a file.
  45 
  46     The default fingerprints types for various fingerprints are shown below:
  47 
  48         AtomPairs              IntSparseIntVect
  49         MACCS166Keys           ExplicitBitVect
  50         Morgan                 UIntSparseIntVect
  51         MorganFeatures         UIntSparseIntVect
  52         PathLength             ExplicitBitVect
  53         TopologicalTorsions    LongSparseIntVect
  54  
  55     The Dice and Tanimoto similarity functions available in RDKit are able to
  56     handle fingerprints corresponding to both IntVect and BitVect. All other
  57     similarity functions, however, expect BitVect fingerprints to calculate
  58     pairwise similarity. Consequently, ExplicitBitVect fingerprints are generated
  59     for AtomPairs, Morgan, MorganFeatures, and TopologicalTorsions for
  60     similarity calculations instead of default IntVect fingerprints.
  61 
  62     The supported input file formats are: SD (.sdf, .sd), SMILES (.smi, .csv, .tsv, .txt)
  63 
  64     The supported output file formats are: SD (.sdf, .sd), SMILES (.smi)
  65 
  66 Options:
  67     -c, --clusteringMethod <Centroid, CLink...>  [default: Centroid]
  68         Clustering method to use for picking a subset of diverse molecules during
  69         hierarchical clustering. Supported values: Centroid, CLink, Gower,
  70         McQuitty, SLink, UPGMA, Ward. This option is ignored for 'MaxMin' value
  71         of '-m, --mode' option. The Clink and SLink corresponding to CompleteLink
  72         and SingleLink cluster method.
  73     -f, --fingerprints <MACCS166Keys, Morgan, PathLength...>  [default: Morgan]
  74         Fingerprints to use for calculating similarity/distance between molecules.
  75         Supported values: AtomPairs, MACCS166Keys, Morgan, MorganFeatures, PathLength,
  76         TopologicalTorsions. The PathLength fingerprints are Daylight like fingerprints.
  77         The Morgan and MorganFeature fingerprints are circular fingerprints, corresponding
  78         Scitegic's Extended Connectivity Fingerprints (ECFP) and Features Connectivity
  79         Fingerprints (FCFP). The values of default parameters for generating fingerprints
  80         can be modified using '-p, --paramsFingerprints' option.
  81     -e, --examples
  82         Print examples.
  83     -h, --help
  84         Print this help message.
  85     -i, --infile <infile>
  86         Input file name.
  87     --infileParams <Name,Value,...>  [default: auto]
  88         A comma delimited list of parameter name and value pairs for reading
  89         molecules from files. The supported parameter names for different file
  90         formats, along with their default values, are shown below:
  91             
  92             SD, MOL: removeHydrogens,yes,sanitize,yes,strictParsing,yes
  93             SMILES: smilesColumn,1,smilesNameColumn,2,smilesDelimiter,space,
  94                 smilesTitleLine,auto,sanitize,yes
  95             
  96         Possible values for smilesDelimiter: space, comma or tab.
  97     -m, --mode <MaxMin or HierarchicalClustering>  [default: MaxMin]
  98         Pick a diverse subset of molecules using MaxMin or hierarchical clustering
  99         methodology.
 100     -n, --numMols <number>  [default: 25]
 101         Number of diverse molecules to pick.
 102     -o, --outfile <outfile>
 103         Output file name.
 104     --outfileParams <Name,Value,...>  [default: auto]
 105         A comma delimited list of parameter name and value pairs for writing
 106         molecules to files. The supported parameter names for different file
 107         formats, along with their default values, are shown below:
 108             
 109             SD: compute2DCoords,auto,kekulize,no
 110             SMILES: kekulize,no,smilesDelimiter,space, smilesIsomeric,yes,
 111                 smilesTitleLine,yes
 112             
 113         Default value for compute2DCoords: yes for SMILES input file; no for all other
 114         file types.
 115     --overwrite
 116         Overwrite existing files.
 117     -p, --paramsFingerprints <Name,Value,...>  [default: auto]
 118         Parameter values to use for generating fingerprints. The default values
 119         are dependent on the value of '-f, --fingerprints' option. In general, it is a
 120         comma delimited list of parameter name and value pairs for the name of
 121         the fingerprints specified using '-f, --fingerprints' option. The supported
 122         parameter names along with their default values for valid fingerprints
 123         names are shown below:
 124             
 125             AtomPairs: minLength,1 ,maxLength,30, useChirality,No
 126             Morgan:   radius,2, useChirality,No
 127             MorganFeatures:   radius,2, useChirality,No
 128             PathLength: minPath,1, maxPath,7, fpSize, 2048, bitsPerHash,2
 129             TopologicalTorsions: useChirality,No
 130             
 131     -s, --similarityMetric <Dice, Tanimoto...>  [default: Tanimoto]
 132         Similarity metric to use for calculating similarity/distance between molecules.
 133         Possible values: BraunBlanquet, Cosine, Dice, Kulczynski, RogotGoldberg,
 134         Russel, Sokal, Tanimoto.
 135     -w, --workingdir <dir>
 136         Location of working directory which defaults to the current directory.
 137 
 138 Examples:
 139     To pick 25 diverse molecules using MaxMin methodology, Tanimoto similarity
 140     metric corresponding to Morgan fingerprints with radius of 2, and write
 141     out a SMILES file, type:
 142 
 143         % RDKitPickDiverseMolecules.py  -i Sample.smi -o SampleOut.smi
 144 
 145     To pick 50 diverse molecules using MaxMin methodology, Dice similarity metric
 146     corresponding to PathLength fingerprints with max path length of 6, and write
 147     out a SD file, type:
 148 
 149         % RDKitPickDiverseMolecules.py  -m MaxMin -f PathLength -s Dice -n 50

 159     To pick 50 diverse molecules using Ward hierarchical methodology methodology,
 160     Dice similarity metric corresponding to MorganFeatures fingerprints with radius

 163         % RDKitPickDiverseMolecules.py  -m HierarchicalClustering -c Ward -n 50
 164           -f MorganFeatures -p 'radius,2,useChirality,No' -i Sample.sdf -o
 165           SampleOut.sdf
 166 
 167     To pick 25 diverse molecules using MaxMin methodology, Tanimoto similarity

 170 
 171         % RDKitPickDiverseMolecules.py  --infileParams
 172           "smilesDelimiter,comma,smilesTitleLine,yes,smilesColumn,1,
 173           smilesNameColumn,2" --outfileParams "compute2DCoords,yes"

 255 def PickDiverseMolecules():
 256     """Pick diverse molecules."""
 257 
 258     Mols = RetrieveMolecules()
 259     MolsFingerprints = GenerateFingerprints(Mols)
 260     DiverseMols = SelectMolecules(Mols, MolsFingerprints)
 261     
 262     WriteMolecules(DiverseMols)
 263 
 264 def SelectMolecules(Mols, MolsFingerprints):
 265     """Select diverse molecules."""
 266 
 267     if OptionsInfo["NumMols"] > len(Mols):
 268         MiscUtil.PrintError("The number of diverse molecules to pick, %d, specified using \"-n, --numMols\" must be less than total number of valid molecules, %d" % (OptionsInfo["NumMols"], len(Mols)))
 269     
 270     DiverseMols = []
 271     if re.match("^MaxMin$", OptionsInfo["Mode"], re.I):
 272         return SelectMoleculesUsingMaxMin(Mols, MolsFingerprints)
 273     elif re.match("^HierarchicalClustering$", OptionsInfo["Mode"], re.I):
 274         return SelectMoleculesUsingHierarchicalClustering(Mols, MolsFingerprints)
 275     else:
 276         MiscUtil.PrintError("The mode vaue, %s, is not a valid mode." % OptionsInfo["Mode"])
 277     
 278     return DiverseMols
 279 
 280 def SelectMoleculesUsingMaxMin(Mols, MolsFingerprints):
 281     """Select diverse molecules using MaxMin methodology."""
 282 
 283     MiscUtil.PrintInfo("\nSelecting diverse molecules using MaxMin methodology and %s similarity metric..." % OptionsInfo["SimilarityMetric"])
 284     
 285     DiverseMols = []
 286     
 287     PoolSize = len(MolsFingerprints)
 288     PickSize = OptionsInfo["NumMols"]
 289     SimilarityFunction = OptionsInfo["SimilarityFunction"]
 290 
 291     Picker = MaxMinPicker()
 292     PairwiseDistance = lambda i, j: 1 - SimilarityFunction(MolsFingerprints[i], MolsFingerprints[j])
 293 
 294     MolIndices = Picker.LazyPick(PairwiseDistance, PoolSize, PickSize)
 295             
 296     for Index in list(MolIndices):
 297         DiverseMols.append(Mols[Index])
 298     
 299     return DiverseMols
 300 
 301 def SelectMoleculesUsingHierarchicalClustering(Mols, MolsFingerprints):
 302     """Select diverse molecules using hierarchical clustering  methodology."""
 303 
 304     try:
 305         import numpy
 306     except ImportError:
 307         MiscUtil.PrintError("Failed to import numpy python module. This is required for picking diverse molecules using hierarchical for clustering.")
 308     
 309     MiscUtil.PrintInfo("\nSelecting diverse molecules using %s hierarchical clustering methodology..." % OptionsInfo["SpecifiedClusteringMethod"])
 310     
 311     DiverseMols = []
 312     
 313     PoolSize = len(MolsFingerprints)
 314     PickSize = OptionsInfo["NumMols"]
 315     DistanceMatrix = GenerateLowerTriangularDistanceMatrix(MolsFingerprints)
 316     
 317     ClusterPicker = HierarchicalClusterPicker(OptionsInfo["SpecifiedClusteringMethodID"])
 318     MolIndices = ClusterPicker.Pick(numpy.asarray(DistanceMatrix), PoolSize, PickSize)
 319     
 320     for Index in MolIndices:
 321         DiverseMols.append(Mols[Index])
 322     
 323     return DiverseMols

 342 def GenerateFingerprints(Mols):
 343     """Generate fingerprints."""
 344 
 345     FingerprintsName = OptionsInfo["SpecifiedFingerprints"]
 346     
 347     MolsFingerprints = []
 348     if re.match("^AtomPairs$", FingerprintsName, re.I):
 349         return GenerateAtomPairsFingerprints(Mols)
 350     elif re.match("^MACCS166Keys$", FingerprintsName, re.I):
 351         return GenerateMACCS166KeysFingerprints(Mols)
 352     elif re.match("^Morgan$", FingerprintsName, re.I):
 353         return GenerateMorganFingerprints(Mols)
 354     elif re.match("^MorganFeatures$", FingerprintsName, re.I):
 355         return GenerateMorganFeaturesFingerprints(Mols)
 356     elif re.match("^PathLength$", FingerprintsName, re.I):
 357         return GeneratePathLengthFingerprints(Mols)
 358     elif re.match("^TopologicalTorsions$", FingerprintsName, re.I):
 359         return GenerateTopologicalTorsionsFingerprints(Mols)
 360     else:
 361         MiscUtil.PrintError("Fingerprints name, %s, is not a valid name" % FingerprintsName)
 362     
 363     return MolsFingerprints

 365 def GenerateAtomPairsFingerprints(Mols):
 366     """Generate AtomPairs fingerprints."""
 367 
 368     MiscUtil.PrintInfo("\nGenerating AtomPairs fingerprints...")
 369     
 370     MinLength = OptionsInfo["FingerprintsParams"]["AtomPairs"]["MinLength"]
 371     MaxLength = OptionsInfo["FingerprintsParams"]["AtomPairs"]["MaxLength"]
 372     UseChirality = OptionsInfo["FingerprintsParams"]["AtomPairs"]["UseChirality"]
 373 
 374     if OptionsInfo["GenerateBitVectFingerints"]:
 375         # Generate ExplicitBitVect fingerprints...
 376         FPSize = 2048
 377         BitsPerHash = 4
 378         MolsFingerprints = [rdMolDescriptors.GetHashedAtomPairFingerprintAsBitVect(Mol, minLength = MinLength, maxLength = MaxLength, includeChirality = UseChirality, nBits = FPSize, nBitsPerEntry = BitsPerHash) for Mol in Mols]
 379     else:
 380         # Generate IntSparseIntVect fingerprints...
 381         MolsFingerprints = [rdMolDescriptors.GetAtomPairFingerprint(Mol, minLength = MinLength, maxLength = MaxLength, includeChirality = UseChirality) for Mol in Mols]
 382 
 383     return MolsFingerprints

 434 def GeneratePathLengthFingerprints(Mols):

 437     MiscUtil.PrintInfo("\nGenerating PathLength fingerprints ...")

 439     MinPath = OptionsInfo["FingerprintsParams"]["PathLength"]["MinPath"]

 469 def GenerateLowerTriangularDistanceMatrix(MolsFingerprints):
 470     """Generate a lower triangular distance matrix without the diagonal."""
 471 
 472     SimilarityFunction = OptionsInfo["SimilarityFunction"]
 473 
 474     DistanceMatrix = []
 475     NumFPs = len(MolsFingerprints)
 476     for Index1 in range(0, NumFPs):
 477         for Index2 in range(0, Index1):
 478             Distance =  1 - SimilarityFunction(MolsFingerprints[Index1], MolsFingerprints[Index2],)
 479             DistanceMatrix.append(Distance)
 480 
 481     return DistanceMatrix

 483 def WriteMolecules(Mols):
 484     """Write out molecules."""
 485 
 486     Outfile = OptionsInfo["Outfile"]
 487     
 488     # Set up a molecule writer...
 489     Writer = None
 490     Writer = RDKitUtil.MoleculesWriter(Outfile, **OptionsInfo["OutfileParams"])
 491     if Writer is None:
 492         MiscUtil.PrintError("Failed to setup a writer for output fie %s " % Outfile)
 493     MiscUtil.PrintInfo("\nGenerating file %s...\n" % Outfile)
 494 
 495     Compute2DCoords = OptionsInfo["OutfileParams"]["Compute2DCoords"]
 496     
 497     # Write out molecules...
 498     for Mol in Mols:
 499         if Compute2DCoords:
 500             AllChem.Compute2DCoords(Mol)
 501         Writer.write(Mol)
 502         
 503     if Writer is not None:
 504         Writer.close()
 505         
 506     MiscUtil.PrintInfo("Total number of diverse molecules selected: %d" % (len(Mols)))

 528 def ProcessSpecifiedFingerprintsName():
 529     """Process specified fingerprints name."""
 530 
 531     #  Set up a canonical fingerprints name map...
 532     CanonicalFingerprintsNamesMap = {}
 533     for Name in OptionsInfo["FingerprintsNames"]:
 534         CanonicalName = Name.lower()
 535         CanonicalFingerprintsNamesMap[CanonicalName] = Name
 536 
 537     # Validate specified fingerprints name...
 538     CanonicalFingerprintsName = OptionsInfo["Fingerprints"].lower()
 539     if CanonicalFingerprintsName not in CanonicalFingerprintsNamesMap:
 540         MiscUtil.PrintError("The fingerprints name, %s, specified using \"-f, --fingerprints\" option is not a valid name." % (OptionsInfo["Fingerprints"]))

 542     OptionsInfo["SpecifiedFingerprints"] = CanonicalFingerprintsNamesMap[CanonicalFingerprintsName]

 544 def ProcessSpecifiedFingerprintsParameters():
 545     """Process specified fingerprints parameters."""
 546 
 547     if re.match("^auto$", OptionsInfo["ParamsFingerprints"], re.I):
 548         # Nothing to process...
 549         return
 550 
 551     SpecifiedFingerprintsName = OptionsInfo["SpecifiedFingerprints"]
 552     
 553     # Parse specified fingerprints parameters...
 554     ParamsFingerprints = re.sub(" ", "", OptionsInfo["ParamsFingerprints"])
 555     if not ParamsFingerprints:
 556         MiscUtil.PrintError("No valid parameter name and value pairs specified using \"-p, --paramsFingerprints\" option corrresponding to fingerprints %s." % (SpecifiedFingerprintsName))
 557 
 558     ParamsFingerprintsWords = ParamsFingerprints.split(",")
 559     if len(ParamsFingerprintsWords) % 2:
 560         MiscUtil.PrintError("The number of comma delimited paramater names and values, %d, specified using \"-p, --paramsFingerprints\" option must be an even number." % (len(ParamsFingerprintsWords)))
 561 
 562     # Setup a canonical parameter names for specified fingerprints...
 563     ValidParamNames = []
 564     CanonicalParamNamesMap = {}
 565     for ParamName in sorted(OptionsInfo["FingerprintsParams"][SpecifiedFingerprintsName]):
 566         ValidParamNames.append(ParamName)
 567         CanonicalParamNamesMap[ParamName.lower()] = ParamName
 568 
 569     # Validate and set paramater names and value...
 570     for Index in range(0, len(ParamsFingerprintsWords), 2):
 571         Name = ParamsFingerprintsWords[Index]
 572         Value = ParamsFingerprintsWords[Index + 1]
 573 
 574         CanonicalName = Name.lower()
 575         if  not CanonicalName in CanonicalParamNamesMap:
 576             MiscUtil.PrintError("The parameter name, %s, specified using \"-p, --paramsFingerprints\" option for fingerprints, %s, is not a valid name. Supported parameter names: %s" % (Name, SpecifiedFingerprintsName, " ".join(ValidParamNames)))
 577 
 578         ParamName = CanonicalParamNamesMap[CanonicalName]
 579         if re.match("^UseChirality$", ParamName, re.I):
 580             if not re.match("^(Yes|No|True|False)$", Value, re.I):
 581                 MiscUtil.PrintError("The parameter value, %s, specified using \"-p, --paramsFingerprints\" option for fingerprints, %s, is not a valid value. Supported values: Yes No True False" % (Value, SpecifiedFingerprintsName))
 582             ParamValue = False
 583             if re.match("^(Yes|True)$", Value, re.I):
 584                 ParamValue = True
 585         else:
 586             ParamValue = int(Value)
 587             if ParamValue <= 0:
 588                 MiscUtil.PrintError("The parameter value, %s, specified using \"-p, --paramsFingerprints\" option for fingerprints, %s, is not a valid value. Supported values: > 0" % (Value, SpecifiedFingerprintsName))
 589         
 590         # Set value...
 591         OptionsInfo["FingerprintsParams"][SpecifiedFingerprintsName][ParamName] = ParamValue
 592 
 593 def ProcessSimilarityMetricParameter():
 594     """Process specified similarity metric value."""
 595 
 596     SimilarityInfoMap = {}
 597     CanonicalNameMap = {}
 598     
 599     for SimilarityFunctionInfo in DataStructs.similarityFunctions:
 600         Name = SimilarityFunctionInfo[0]
 601         Function = SimilarityFunctionInfo[1]
 602         
 603         SimilarityInfoMap[Name] = Function
 604         CanonicalName = Name.lower()
 605         CanonicalNameMap[CanonicalName] = Name
 606     
 607     SpecifiedCanonicalName = OptionsInfo["SimilarityMetric"].lower()
 608     SimilarityFunction = None
 609     if  SpecifiedCanonicalName in CanonicalNameMap:
 610         SimilarityName = CanonicalNameMap[SpecifiedCanonicalName]
 611         SimilarityFunction = SimilarityInfoMap[SimilarityName]
 612     else:
 613         MiscUtil.PrintError("Similarity metric name, %s, is not a valid name. " % OptionsInfo["SimilarityMetric"])
 614         
 615     OptionsInfo["SimilarityMetric"] = SimilarityName
 616     OptionsInfo["SimilarityFunction"] = SimilarityFunction
 617 
 618     # RDKit similarity functions, besides Dice and Tanimoto, are not able to handle int bit vectors...
 619     GenerateBitVectFingerints = False
 620     if not re.match("^(Tanimoto|Dice)$", SimilarityName, re.I):
 621         GenerateBitVectFingerints = True
 622     OptionsInfo["GenerateBitVectFingerints"] = GenerateBitVectFingerints

 624 def ProcessClusteringMethodParameter():
 625     """Process specified clustering method parameter."""
 626 
 627     OptionsInfo["SpecifiedClusteringMethod"] = ""
 628     OptionsInfo["SpecifiedClusteringMethodID"] = ""
 629     
 630     if not re.match("^HierarchicalClustering$", OptionsInfo["Mode"], re.I):
 631         # Nothing to process...
 632         return
 633 
 634     # Setup a canonical cluster method name map..
 635     ClusteringMethodInfoMap = {}
 636     CanonicalClusteringMethodNameMap = {}
 637     for Name in sorted(rdSimDivPickers.ClusterMethod.names):
 638         NameID =  rdSimDivPickers.ClusterMethod.names[Name]
 639         ClusteringMethodInfoMap[Name] = NameID
 640         
 641         CanonicalName = Name.lower()
 642         CanonicalClusteringMethodNameMap[CanonicalName] = Name
 643 
 644     CanonicalName = OptionsInfo["ClusteringMethod"].lower()
 645     if not CanonicalName in CanonicalClusteringMethodNameMap:
 646         MiscUtil.PrintError("The cluster method, %s, specified using \"-c, --clusteringMethod\" option is not a valid name." % (OptionsInfo["ClusteringMethod"]))
 647 
 648     SpecifiedClusteringMethodName = CanonicalClusteringMethodNameMap[CanonicalName]
 649     OptionsInfo["SpecifiedClusteringMethod"] = SpecifiedClusteringMethodName
 650     OptionsInfo["SpecifiedClusteringMethodID"] = ClusteringMethodInfoMap[SpecifiedClusteringMethodName]
