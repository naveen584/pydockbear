  29 """
  30 RDKitSearchFunctionalGroups.py - Search for functional groups using SMARTS patterns
  31 
  32 Usage:
  33     RDKitSearchFunctionalGroups.py  [--combineMatches <yes or no>] [--combineOperator <and or or>]
  34                                            [--groupNamesFile <FileName or auto>] [--infileParams <Name,Value,...>]
  35                                            [--mode <retrieve or count>] [--negate <yes or no>]
  36                                            [--outfileParams <Name,Value,...>] [--overwrite] [--useChirality <yes or no>]
  37                                            [-w <dir>] [-o <outfile>] -i <infile> -f <Name1,Name2,Name3... or All>
  38     RDKitSearchFunctionalGroups.py [--groupNamesFile <FileName or auto>] -l | --list
  39     RDKitSearchFunctionalGroups.py -h | --help | -e | --examples
  40 
  41 Description:
  42     Perform a substructure search in an input file using SMARTS patterns for functional
  43     groups and write out the matched molecules to an output file or simply count the
  44     number of matches.
  45 
  46     The SMARTS patterns for specified functional group(s) are retrieved from file,
  47     Functional_Group_Hierarchy.txt, available in RDKit data directory.
  48 
  49     The names of valid functional groups and hierarchies  are dynamically retrieved from the
  50     functional groups hierarchy file and are shown below:
  51 
  52         AcidChloride, AcidChloride.Aromatic, AcidChloride.Aliphatic
  53         Alcohol, Alcohol.Aromatic, Alcohol.Aliphatic
  54         Aldehyde, Aldehyde.Aromatic, Aldehyde.Aliphatic
  55         Amine, Amine.Primary, Amine.Primary.Aromatic, Amine.Primary.Aliphatic,
  56         Amine.Secondary, Amine.Secondary.Aromatic, Amine.Secondary.Aliphatic
  57         Amine.Tertiary, Amine.Tertiary.Aromatic, Amine.Tertiary.Aliphatic
  58         Amine.Aromatic, Amine.Aliphatic, Amine.Cyclic
  59         Azide, Azide.Aromatic, Azide.Aliphatic
  60         BoronicAcid, BoronicAcid.Aromatic, BoronicAcid.Aliphatic
  61         CarboxylicAcid, CarboxylicAcid.Aromatic, CarboxylicAcid.Aliphatic,
  62         CarboxylicAcid.AlphaAmino
  63         Halogen, Halogen.Aromatic, Halogen.Aliphatic
  64         Halogen.NotFluorine, Halogen.NotFluorine.Aliphatic,
  65         Halogen.NotFluorine.Aromatic
  66         Halogen.Bromine, Halogen.Bromine.Aliphatic, Halogen.Bromine.Aromatic,
  67         Halogen.Bromine.BromoKetone
  68         Isocyanate, Isocyanate.Aromatic, Isocyanate.Aliphatic
  69         Nitro, Nitro.Aromatic, Nitro.Aliphatic,
  70         SulfonylChloride, SulfonylChloride.Aromatic, SulfonylChloride.Aliphatic
  71         TerminalAlkyne
  72 
  73     The supported input file formats are: SD (.sdf, .sd), SMILES (.smi, .csv, .tsv, .txt)
  74 
  75     The supported output file formats are: SD (.sdf, .sd), SMILES (.smi)
  76 
  77 Options:
  78     -c, --combineMatches <yes or no>  [default: yes]
  79         Combine search results for matching SMARTS patterns of specified functional groups
  80         against a molecule. Possible values: yes or no.
  81         
  82         The matched molecules are written to a single output file for "yes" value. Otherwise,
  83         multiple output files are generated, one for each functional group. The names of  
  84         these files correspond to a combination of the basename of the specified output file
  85         and the name of the functional group.
  86         
  87         No output files are generated during "count" value of "-m, --mode" option.
  88     --combineOperator <and or or>  [default: and]
  89         Logical operator to use for combining match results corresponding to specified
  90         functional group names before writing out a single file. This option is ignored
  91         during "No" value of  "-c, --combineMatches" option.
  92     -e, --examples
  93         Print examples.
  94     -g, --groupNamesFile <FileName or auto>  [default: auto]
  95         Specify a file name containing data for functional groups hierarchy or use functional
  96         group hierarchy file, Functional_Group_Hierarchy.txt, available in RDKit data directory.
  97         
  98         RDKit data format: Name<tab>Smarts<tab>Label<tab>RemovalReaction (optional)
  99         
 100         The format of data in local functional group hierarchy must match format of the
 101         data in functional group file available in RDKit data directory.
 102     -f, --functionalGroups <Name1,Name2,Name3... or All>  [default: none]
 103         Functional group names for performing substructure SMARTS search. Possible values:
 104         Comma delimited list of valid functional group names or All. The current set of valid
 105         functional group names are listed in the description section.
 106         
 107         The match results for multiple functional group names are combined using 'and'
 108         operator before writing them out to single file. No merging of match results takes
 109         place during generation of individual result files corresponding to fictional group
 110         names.
 111         
 112         The functional group name may be started with an exclamation mark to negate
 113         the match result for that fictional group.
 114     -h, --help
 115         Print this help message.
 116     -i, --infile <infile>
 117         Input file name.
 118     --infileParams <Name,Value,...>  [default: auto]
 119         A comma delimited list of parameter name and value pairs for reading
 120         molecules from files. The supported parameter names for different file
 121         formats, along with their default values, are shown below:
 122             
 123             SD, MOL: removeHydrogens,yes,sanitize,yes,strictParsing,yes
 124             SMILES: smilesColumn,1,smilesNameColumn,2,smilesDelimiter,space,
 125                 smilesTitleLine,auto,sanitize,yes
 126             
 127         Possible values for smilesDelimiter: space, comma or tab.
 128     -l, --list
 129         List functional groups information without performing any search.
 130     -m, --mode <retrieve or count>  [default: retrieve]
 131         Specify whether to retrieve and write out matched molecules to an output
 132         file or simply count the number of matches.
 133     -o, --outfile <outfile>
 134         Output file name.
 135     --outfileParams <Name,Value,...>  [default: auto]
 136         A comma delimited list of parameter name and value pairs for writing
 137         molecules to files. The supported parameter names for different file
 138         formats, along with their default values, are shown below:
 139             
 140             SD: compute2DCoords,auto,kekulize,no
 141             SMILES: kekulize,no,smilesDelimiter,space, smilesIsomeric,yes,
 142                 smilesTitleLine,yes
 143             
 144         Default value for compute2DCoords: yes for SMILES input file; no for all other
 145         file types.
 146     --overwrite
 147         Overwrite existing files.
 148     -u, --useChirality <yes or no>  [default: no]
 149         Use stereochemistry information for SMARTS search.
 150     -w, --workingdir <dir>
 151         Location of working directory which defaults to the current directory.
 152 
 153 Examples:
 154     To list names of all available functional groups along with their SMARTS patterns, type:
 155 
 156         % RDKitSearchFunctionalGroups.py -l
 157 
 158     To retrieve molecule containing amine functional group and write out a SMILES file, type: 
 159 
 160         % RDKitSearchFunctionalGroups.py -f Amine -i Sample.smi -o SampleOut.smi
 161 
 162     To retrieve molecules containing amine functional group but not halogens and carboxylic
 163     acid functional groups and write out a SMILES file, type: 
 164 
 165         % RDKitSearchFunctionalGroups.py -f 'Amine,!Halogen,!CarboxylicAcid'
 166           -i Sample.smi -o SampleOut.smi
 167 
 168     To retrieve molecules containing amine, halogens or carboxylic  acid functional groups
 169     and write out a SMILES file, type: 
 170 
 171         % RDKitSearchFunctionalGroups.py -f 'Amine,Halogen,CarboxylicAcid'
 172           --combineOperator or -i Sample.smi -o SampleOut.smi
 173 
 174     To retrieve molecules containing amine and carboxylic acid functional groups defined in
 175     a local functional groups hierarchy file and write out individual SD files for each
 176     funcitonal group, type: 
 177 
 178         % RDKitSearchFunctionalGroups.py -f 'Amine,CarboxylicAcid' -i Sample.sdf 
 179           -g Custom_Functional_Group_Hierarchy.txt --combineMatches No -o SampleOut.sdf
 180 
 181     To count number of all functional groups in molecules without writing out an output
 182     files, type:
 183 
 184         % RDKitSearchFunctionalGroups.py -m count -f All --combineMatches no -i Sample.smi
 185 
 186     To retrieve molecule not containing aromatic alcohol and aromatic halogen functional

 282 def PerformSearch():
 283     """Perform search using SMARTS pattern for specified functional groups."""
 284 
 285     Infile = OptionsInfo["Infile"]
 286     Outfile = OptionsInfo["Outfile"]
 287 
 288     Groups = OptionsInfo["SpecifiedFunctionalGroups"]
 289     GroupsNegateMatch = OptionsInfo["SpecifiedFunctionalGroupsNegateMatch"]
 290     GroupsPatternMols = OptionsInfo["SpecifiedFunctionalGroupsPatternMols"]
 291     
 292     GroupsOutfiles = OptionsInfo["SpecifiedFunctionalGroupsOutfiles"]
 293     GroupsCount = len(Groups)
 294 
 295     CombineMatchResults = OptionsInfo["CombineMatchResults"]
 296     AndCombineOperatorMode = OptionsInfo["AndCombineOperatorMode"]
 297     
 298     CountMode = OptionsInfo["CountMode"]
 299     UseChirality = OptionsInfo["UseChirality"]
 300 
 301     # Setup a molecule reader...
 302     MiscUtil.PrintInfo("\nProcessing file %s..." % Infile)
 303     Mols  = RDKitUtil.ReadMolecules(Infile, **OptionsInfo["InfileParams"])
 304     
 305     # Set up  molecule writers...
 306     Writer = None
 307     GroupOutfilesWriters = []
 308     if not CountMode:
 309         Writer,  GroupOutfilesWriters = SetupMoleculeWriters(CombineMatchResults, Outfile, GroupsOutfiles)
 310     
 311     # Initialize pattern mols match count and status...
 312     GroupsPatternMolsMatchCount = [0] * GroupsCount
 313     GroupsPatternMolsMatchStatus = [False] * GroupsCount
 314 
 315     # Process molecules...
 316     MolCount = 0
 317     ValidMolCount = 0
 318     MatchCount = 0
 319 
 320     Compute2DCoords = OptionsInfo["OutfileParams"]["Compute2DCoords"]
 321     
 322     for Mol in Mols:
 323         MolCount += 1
 324         
 325         if Mol is None:
 326             continue
 327         
 328         if RDKitUtil.IsMolEmpty(Mol):
 329             MolName = RDKitUtil.GetMolName(Mol, MolCount)
 330             MiscUtil.PrintWarning("Ignoring empty molecule: %s" % MolName)
 331             continue
 332         
 333         ValidMolCount += 1
 334 
 335         # Match pattern mols...
 336         for GroupIndex in range(0, GroupsCount):
 337             GroupsPatternMolsMatchStatus[GroupIndex] = DoesPatternMolMatch(GroupsPatternMols[GroupIndex], Mol, UseChirality, GroupsNegateMatch[GroupIndex])
 338             if GroupsPatternMolsMatchStatus[GroupIndex]:
 339                 GroupsPatternMolsMatchCount[GroupIndex] += 1
 340         
 341         # Match mol against all specified criteria...
 342         MolMatched = DoesMolMeetSpecifiedMatchCriteria(GroupsPatternMolsMatchStatus, CombineMatchResults, AndCombineOperatorMode)
 343         if MolMatched:
 344             MatchCount += 1
 345             
 346         # Nothing to write...
 347         if CountMode or (not MolMatched):
 348             continue
 349         
 350         # Write out matched molecules...
 351         if Compute2DCoords:
 352             AllChem.Compute2DCoords(Mol)
 353         
 354         if CombineMatchResults:
 355             Writer.write(Mol)
 356         else:
 357             for GroupIndex in range(0, GroupsCount):
 358                 if GroupsPatternMolsMatchStatus[GroupIndex]:
 359                     GroupOutfilesWriters[GroupIndex].write(Mol)
 360     
 361     if Writer is not None:
 362         Writer.close()
 363     for GroupOutfileWriter in GroupOutfilesWriters:
 364         GroupOutfileWriter.close()
 365         
 366     MiscUtil.PrintInfo("\nTotal number of molecules: %d" % MolCount)
 367     MiscUtil.PrintInfo("Number of valid molecules: %d" % ValidMolCount)
 368     MiscUtil.PrintInfo("Number of ignored molecules: %d" % (MolCount - ValidMolCount))
 369 
 370     MiscUtil.PrintInfo("\nTotal number of molecules matched against specified match criteria: %d" % MatchCount)
 371     
 372     MiscUtil.PrintInfo("\nNumber of molecuels matched against individual functional groups:")
 373     MiscUtil.PrintInfo("FunctionalGroupName,MatchCount")
 374     
 375     for GroupIndex in range(0, GroupsCount):
 376         GroupName = Groups[GroupIndex]
 377         if GroupsNegateMatch[GroupIndex]:
 378             GroupName = '!' + GroupName
 379         GroupMatchCount = GroupsPatternMolsMatchCount[GroupIndex]
 380         MiscUtil.PrintInfo("%s,%d" % (GroupName, GroupMatchCount))
 381 
 382 def DoesMolMeetSpecifiedMatchCriteria(GroupsPatternMolsMatchStatus,  CombineMatchResults, AndCombineOperatorMode):
 383     """Match molecule using specified match criteia."""
 384 
 385     if CombineMatchResults and AndCombineOperatorMode:
 386         # Must match all specified SMARTS
 387         Status = True
 388         for MatchStatus in GroupsPatternMolsMatchStatus:
 389             if not MatchStatus:
 390                 Status = False
 391                 break
 392     else:
 393         # One match is enough...
 394         Status = False
 395         for MatchStatus in GroupsPatternMolsMatchStatus:
 396             if MatchStatus:
 397                 Status = True
 398                 break
 399     
 400     return Status
 401     
 402 def SetupMoleculeWriters(CombineMatchResults, Outfile, GroupsOutfiles):
 403     """Set up molecule writers for output files."""
 404 
 405     Writer = None
 406     GroupOutfilesWriters = []
 407     
 408     if CombineMatchResults:
 409         Writer = RDKitUtil.MoleculesWriter(Outfile, **OptionsInfo["OutfileParams"])
 410         if Writer is None:
 411             MiscUtil.PrintError("Failed to setup a writer for output fie %s " % Outfile)
 412         MiscUtil.PrintInfo("Generating file %s..." % Outfile)
 413     else:
 414         for GroupOutfile in GroupsOutfiles:
 415             GroupOutfileWriter = RDKitUtil.MoleculesWriter(GroupOutfile, **OptionsInfo["OutfileParams"])
 416             if GroupOutfileWriter is None:
 417                 MiscUtil.PrintError("Failed to setup a writer for output fie %s " % Writer)
 418             GroupOutfilesWriters.append(GroupOutfileWriter)
 419         
 420         GroupsCount = len(GroupsOutfiles)
 421         if GroupsCount > 4:
 422             MiscUtil.PrintInfo("Generating %d output files with the following file name format: %s<GroupName>.%s" % (GroupsCount, OptionsInfo["OutfileBasename"], OptionsInfo["OutfileExt"]))
 423         else:
 424             Delmiter = ', '
 425             OutfileNames = Delmiter.join(GroupsOutfiles)
 426             MiscUtil.PrintInfo("Generating %d output files: %s..." % (GroupsCount, OutfileNames))
 427 
 428     return (Writer, GroupOutfilesWriters)
 429     
 430 def DoesPatternMolMatch(PatternMol, Mol, UseChirality, NegateMatch):
 431     """Perform a substructure match for the presence of pattern molecule in a molecule."""
 432 
 433     MolMatched = Mol.HasSubstructMatch(PatternMol, useChirality = UseChirality)
 434     if NegateMatch:
 435         if MolMatched:
 436             MolMatched = False
 437         else:
 438             MolMatched = True
 439     
 440     return MolMatched

 442 def ProcessFunctionalGroupsInfo():
 443     """Process functional groups information."""
 444     
 445     RetrieveFunctionalGroupsInfo()
 446     ProcessSpecifiedFunctionalGroups()
 447 
 448     SetupFunctionalGroupsSMARTSPatterns()
 449     SetupFunctionalGroupsOutputFileNames()
 450     
 451 def ProcessSpecifiedFunctionalGroups():
 452     """Process and validate specified functional groups"""
 453     
 454     OptionsInfo["SpecifiedFunctionalGroups"] = []
 455     OptionsInfo["SpecifiedFunctionalGroupsNegateMatch"] = []
 456     
 457     if re.match("^All$", OptionsInfo["FunctionalGroups"], re.I):
 458         OptionsInfo["SpecifiedFunctionalGroups"] = FunctionalGroupsMap['Names']
 459         OptionsInfo["SpecifiedFunctionalGroupsNegateMatch"] = [False] * len(OptionsInfo["SpecifiedFunctionalGroups"])
 460         return
 461     
 462     # Set up a map of valid group names for checking specified group names...
 463     CanonicalGroupNameMap = {}
 464     for GroupName in FunctionalGroupsMap['Names']:
 465         CanonicalGroupNameMap[GroupName.lower()] = GroupName
 466         
 467     # Parse and validate specified names...
 468     GroupNames = re.sub(" ", "", OptionsInfo["FunctionalGroups"])
 469     if not GroupNames:
 470         MiscUtil.PrintError("No functional group name specified for \"-f, --functionalGroups\" option")
 471     
 472     SpecifiedFunctionalGroups = []
 473     SpecifiedNegateMatchStatus = []
 474     
 475     for GroupName in GroupNames.split(","):
 476         CanonicalGroupName = GroupName.lower()
 477         NegateMatchStatus = False
 478         if re.match("^!", CanonicalGroupName, re.I):
 479             NegateMatchStatus = True
 480             CanonicalGroupName = re.sub("^!", "", CanonicalGroupName)
 481         if CanonicalGroupName in CanonicalGroupNameMap:
 482             SpecifiedFunctionalGroups.append(CanonicalGroupNameMap[CanonicalGroupName])
 483             SpecifiedNegateMatchStatus.append(NegateMatchStatus)
 484         else:
 485             MiscUtil.PrintWarning("The functional group name, %s, specified using \"-f, --functionalGroups\" option is not a valid name." % (GroupName))
 486 
 487     if not len(SpecifiedFunctionalGroups):
 488         MiscUtil.PrintError("No valid functional group names specified for \"-f, --functionalGroups\" option")
 489         
 490     OptionsInfo["SpecifiedFunctionalGroups"] = SpecifiedFunctionalGroups
 491     OptionsInfo["SpecifiedFunctionalGroupsNegateMatch"] = SpecifiedNegateMatchStatus
 492 
 493 def SetupFunctionalGroupsSMARTSPatterns():
 494     """Setup SMARTS patterns for specified functional groups."""
 495 
 496     OptionsInfo["SpecifiedFunctionalGroupsSMARTSPatterns"] = []
 497     OptionsInfo["SpecifiedFunctionalGroupsPatternMols"] = []
 498     
 499     for Name in OptionsInfo["SpecifiedFunctionalGroups"]:
 500         SMARTSPattern = FunctionalGroupsMap['SMARTSPattern'][Name]
 501         PatternMol = Chem.MolFromSmarts(SMARTSPattern)
 502         if PatternMol is None:
 503             MiscUtil.PrintError("Failed to parse SMARTS pattern, %s, for function group, %s" % (SMARTSPattern, Name))
 504         
 505         OptionsInfo["SpecifiedFunctionalGroupsSMARTSPatterns"].append(SMARTSPattern)
 506         OptionsInfo["SpecifiedFunctionalGroupsPatternMols"].append(PatternMol)
 507 
 508 def SetupFunctionalGroupsOutputFileNames():
 509     """Setup output file names for specified functional group names."""
 510 
 511     OptionsInfo["SpecifiedFunctionalGroupsOutfiles"] = []
 512     
 513     if OptionsInfo["CountMode"]:
 514         # No need of any output file...
 515         return
 516     
 517     if OptionsInfo["CombineMatchResults"]:
 518         # No need of output files for specified functional groups...
 519         return
 520     
 521     OutfileBasename = OptionsInfo["OutfileBasename"]
 522     OutfileExt = OptionsInfo["OutfileExt"]
 523     SpecifiedFunctionalGroupsOutfiles = []
 524 
 525     GroupsCount = len(OptionsInfo["SpecifiedFunctionalGroups"])
 526     for GroupIndex in range(0, GroupsCount):
 527         GroupName = OptionsInfo["SpecifiedFunctionalGroups"][GroupIndex]
 528         if OptionsInfo["SpecifiedFunctionalGroupsNegateMatch"][GroupIndex]:
 529             GroupName = "Not" + GroupName
 530         GroupName = re.sub("\.", "", GroupName)
 531         
 532         GroupOutfile = "%s%s.%s" % (OutfileBasename, GroupName, OutfileExt)
 533         SpecifiedFunctionalGroupsOutfiles.append(GroupOutfile)
 534 
 535     OptionsInfo["SpecifiedFunctionalGroupsOutfiles"] = SpecifiedFunctionalGroupsOutfiles
 536     
 537 def RetrieveFunctionalGroupsInfo():
 538     """Retrieve functional groups information"""
 539 
 540     MiscUtil.PrintInfo("\nRetrieving data from default RDKit functional groups hierarchy file Functional_Group_Hierarchy.txt...")
 541     
 542     FunctionalGroupNamesFile = OptionsInfo["GroupNamesFile"]
 543     FunctionalGroupsNodes = FunctionalGroups.BuildFuncGroupHierarchy(FunctionalGroupNamesFile)
 544 
 545     FunctionalGroupsMap['Names'] = []
 546     FunctionalGroupsMap['SMARTSPattern'] = {}
 547     
 548     RetrieveDataFromFunctionalGroupsHierarchy(FunctionalGroupsNodes)
 549 
 550     if not len(FunctionalGroupsMap['Names']):
 551         MiscUtil.PrintError("Failed to retrieve any functional group names and SMARTS patterns...")
 552         
 553     MiscUtil.PrintInfo("Total number of functional groups present functional group hierarchy: %d" % (len(FunctionalGroupsMap['Names'])))
 554     
 555 def RetrieveDataFromFunctionalGroupsHierarchy(FGNodes):
 556     """Retrieve functional groups data by recursively visiting functional group nodes."""
 557     
 558     for FGNode in FGNodes:
 559         Name = FGNode.label
 560         SMARTSPattern = FGNode.smarts
 561 
 562         if Name in FunctionalGroupsMap['SMARTSPattern']:
 563             MiscUtil.PrintWarning("Ignoring duplicate functional group name: %s..." % Name)
 564         else:
 565             FunctionalGroupsMap['Names'].append(Name)
 566             FunctionalGroupsMap['SMARTSPattern'][Name] = SMARTSPattern
 567 
 568         RetrieveDataFromFunctionalGroupsHierarchy(FGNode.children)
 569 
 570 def ListFunctionalGroupsInfo():
 571     """List functional groups information"""
 572 
 573     MiscUtil.PrintInfo("\nListing available functional groups names and SMARTS patterns...")
 574     MiscUtil.PrintInfo("\nFunctionalGroupName\tSMARTSPattern")
 575     
 576     for Name in sorted(FunctionalGroupsMap['Names']):
 577         SMARTSPattern = FunctionalGroupsMap['SMARTSPattern'][Name]
 578         MiscUtil.PrintInfo("%s\t%s" % (Name, SMARTSPattern))
 579     
 580     MiscUtil.PrintInfo("")

 582 def ProcessOptions():
 583     """Process and validate command line arguments and options"""
 584     
 585     MiscUtil.PrintInfo("Processing options...")

 587     # Validate options...
 588     ValidateOptions()

 591

 596             FileDir, FileName, FileExt = MiscUtil.ParseFileName(Options["--outfile"])

 598             OptionsInfo["OutfileExt"] = FileExt
